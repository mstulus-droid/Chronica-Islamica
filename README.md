# Chronica Islamica

Linimasa interaktif sejarah Islam klasik dengan pendekatan historis-sekuler dan non-apologis.

## Isi

- Tokoh, penguasa, panglima, ilmuwan/ulama, dan peristiwa penting.
- Tampilan timeline interaktif dalam satu file HTML.
- Favicon dan identitas visual aplikasi.

## Menjalankan

Buka `index.html` langsung di browser.
